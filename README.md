## Projects

### [Kronos](https://github.com/biscuitrescue/kronos)  
![Zig](https://img.shields.io/badge/-Zig-F7A41D?logo=zig&logoColor=black)  
A high-performance, deterministic filesystem time-machine and data debugger. Built with Merkle-DAGs and Content-Defined Chunking to enable infinite undo and granular data provenance.

### [Gatekeep](https://github.com/biscuitrescue/gatekeep)  
![Rust](https://img.shields.io/badge/-Rust-black?logo=rust&logoColor=white)  
GitHub-style access control for SSH servers. Define `authorized_keys` in TOML, track changes via Git, and sync via a secure daemon. Built from scratch in Rust.
